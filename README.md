# CustomsInsight
CustomsInsight 是一个专门为财务和会计人员设计的自动化工具。它能自动监控本地文件夹，利用 AI 和规则算法解析 PDF 格式的《中华人民共和国海关出口货物报关单》，并将提取的结构化数据精准地映射到 Excel 出口明细台账中，彻底告别繁琐的人工录入。
✨ 核心特性
📂 文件夹自动化监控：采用 watchdog 实时监听，只需将下载的 PDF 拖入文件夹即可触发解析。

🔍 深度数据提取：基于 pdfplumber 精准抓取报关单号、合同号、成交方式、货物明细等 50+ 关键字段。

⚖️ 智能费用分摊：支持 FOB/C&F/CIF 等贸易术语逻辑，自动按货值比例分摊运费和保费。

📊 一单多货支持：完美处理一张报关单包含多个商品项的情况，自动在 Excel 中生成多行记录。

🛡️ 数据校验机制：内置重复申报检查，防止同一份报关单多次录入台账。

🔒 本地化处理：所有解析逻辑均在本地完成，无需上传云端，确保财务数据绝对安全。

🛠 技术架构
语言: Python 3.10+

PDF 解析: pdfplumber

数据处理: Pandas

Excel 驱动: openpyxl

文件监控: watchdog
